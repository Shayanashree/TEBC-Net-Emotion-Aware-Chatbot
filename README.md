# TEBC-Net: Emotion-Aware Conversational Agent

This project presents an AI-based chatbot that detects user emotions using both **text and facial expressions**.  
The system combines **Natural Language Processing (BERT)** and **Computer Vision (CNN)** to create a more empathetic and intelligent conversational agent.

##  Features
- Detects emotions from **text input** using BERT.
- Detects emotions from **facial expressions** using CNN.
- Multimodal emotion analysis for better accuracy.
- Generates emotion-aware responses.
- Interactive web interface.
